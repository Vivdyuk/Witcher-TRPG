# Witcher-TRPG

Маленька тулза для полегшення життя при створенні персонажу і рольовій грі "Відьмак".
## По пунктах
- Завантажити ексель файл на Ґугл диск
- Відкрити скрипт едітор (Розшиорення -> Apps Script)
- Замінити основний файл на **copyMe.js** (вміст)
- На шиті задати Расу і Професію
- Запустити скрипт
